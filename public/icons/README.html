Instructions pour générer les icônes PWA

Pour résoudre l'erreur 404 sur les icônes PWA, vous devez générer les icônes suivantes :

  • /public/icons/icon-72x72.png
  • /public/icons/icon-96x96.png
  • /public/icons/icon-128x128.png
  • /public/icons/icon-144x144.png (celui qui cause l'erreur 404)
  • /public/icons/icon-152x152.png
  • /public/icons/icon-192x192.png
  • /public/icons/icon-384x384.png
  • /public/icons/icon-512x512.png

Option 1 : Utiliser le script Node.js

Installez Sharp et exécutez le script scripts/generate-pwa-icons.js :

npm install sharp
node scripts/generate-pwa-icons.js

Option 2 : Utiliser un générateur en ligne

Vous pouvez utiliser un générateur d'icônes PWA en ligne comme PWA Image Generator ou App Manifest Generator.

Option 3 : Solution temporaire

Si vous voulez juste faire disparaître l'erreur 404, créez manuellement un fichier PNG de 144x144 pixels et placez-le dans /public/icons/icon-144x144.png.
